# Module 09: UI/UX Design Specification

> 所属 PRD：ruff-prd-v3.md
> 优先级：P0

## 1. 概述

本模块定义 Ruff 的 UI/UX 设计规范，使用 SwiftUI via @expo/ui 构建界面。

## 2. 设计原则

1. **极简优先**：每个页面只展示最必要的信息
2. **操作便捷**：核心操作一步完成，打卡流程不超过 3 次点击
3. **反馈明确**：每个操作都有明确的状态反馈
4. **离线可用**：无网络时核心功能不受影响

## 9. 深色模式

使用 SwiftUI 的 `ColorScheme` 适配深色模式：

```swift
struct ContentView: View {
    @Environment(\.colorScheme) var colorScheme

    var body: some View {
        ZStack {
            Color(colorScheme == .dark ? "#121212" : "#FFFBFF")
        }
    }
}
```

## 10. 交互规范

### 10.1 动画

| 交互 | 动画 |
|------|------|
| Tab 切换 | 300ms ease-in-out |
| 卡片展开 | 200ms ease-out |
| 按钮按下 | 100ms scale(0.98) |

|------|------|

| 右滑 | 查看后一天 |
